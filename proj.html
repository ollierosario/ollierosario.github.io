name

Name, my capstone project for the minor in Applied Music Technology, is an electronic instrument driven by gestures. It uses two ultrasonic distance sensors and an Adafruit APDS9960 breakout board. The breakout board is capable of tracking gestures, color, and distance. Currently, only the gesture tracking is in use, detecting left, right, up, and downward motions. When these are reported to the software, sound is triggered. The different motions control the timbre of sound, changing the constants input to my syntehsizer. The ultrasonic distance sensors, positioned on opposite sides of the instrument, are used to measure how far away the performer's hand is. This distance is then transformed into pitches. The program stores the most recent pitch, and outputs it through the desired synthesis when a gesture is detected.
building the casing
  One of the extended goals of this project is to implement multiple synthesizers and create an "all purpose string instrument." This would use different gestures to switch between bowed string synthesis and plucked string synthesis, and potentially use the distance tracking capabilities of the APDS to trigger hammered string timbres.
Arduino and sensors

  To assemble the hardware, I learned how to solder and desolder. The ultrasonic sensors had to have the pins removed and swapped with new ones so they would point upward. I wired the components to a shield so the components can be easily removed from the Arduino. In designing the casing, I considered multiple options, including making the entire thing out of wood. Aesthetically, I wanted the instrument to appear like a dulcimer, so I decided to make the casing out of a decorative box. I printed my design so i could cut it cleanly, and I'm quite pleased with the result. It is supported with wooden blocks and pegs, two of which keep the electronics, which are attached to a sheet of craft board, in line with the lid.

storytelling

storytelling is a visual project concerning perceived limitations of print media. Through use of embroidery, watercolor, collage, and more, I create an abstract story. The plot is highly open, rendering the viewer able to interpret the project however they see fit. While still ongoing, the end goal is to put together a small book of these illustrations, with my personal narrative in mind, but without words.

At this point, I am finished with the pages present here. I draw inspiration from magical realism and childhood memories of reading fantasy novels.
